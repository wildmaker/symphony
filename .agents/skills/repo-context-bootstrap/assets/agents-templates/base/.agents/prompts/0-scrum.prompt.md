# Scrum

## Goal

Convert a broad request into ordered, reviewable tasks in `.agents/TaskLogs/Scrum.md`.

## Write Scope

Only update `.agents/TaskLogs/Scrum.md` unless the user explicitly asks for implementation.

## Required Sections

- `# !!!SCRUM!!!`
- `# DESIGN REQUEST`
- `# UPDATES`
- `# TASKS`
- `# IMPACT TO THE KNOWLEDGE BASE`
- `# !!!FINISHED!!!`

## Process

1. Read `.agents/profile.md`, `.agents/WORKFLOW.md`, and relevant KnowledgeBase entries.
2. Preserve newer user updates in `# UPDATES`.
3. Split broad work into tasks that are independently understandable and verifiable.
4. For each task, include what to do, rationale, and acceptance signal.
5. Mark whether a durable KnowledgeBase or Learning update is expected.
6. End with `# !!!FINISHED!!!`.
