# Summarizing

## Goal

Convert the planning artifact into an executable checklist in `.agents/TaskLogs/Execution.md`.

## Write Scope

Only update `.agents/TaskLogs/Execution.md` unless the user explicitly asks for implementation.

## Process

1. Read `.agents/TaskLogs/Planning.md`.
2. Preserve affected projects and verification targets.
3. Convert each plan step into a TODO/DONE execution step.
4. Include the evidence needed for each step.
5. Prepare a `# FIXING ATTEMPTS` section.
6. End with `# !!!FINISHED!!!`.
