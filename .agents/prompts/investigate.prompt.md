# Investigate

## Goal

Use this when the root cause is unknown or the task needs evidence before planning a fix.

## Write Scope

Prefer `.agents/TaskLogs/Investigate.md` until a proposal is confirmed.

## Process

1. Reproduce or capture the current signal.
2. Record observations, logs, traces, screenshots, or command output.
3. Propose multiple plausible causes/fixes when evidence is incomplete.
4. Confirm or deny each proposal with evidence.
5. Preserve the confirmed proposal and verification path for planning/execution.
