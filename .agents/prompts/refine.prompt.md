# Refine Learning

## Goal

Extract reusable lessons from completed task logs and verification failures.

## Process

1. Read relevant task logs and failure/fix attempts.
2. Keep only lessons that are actionable, likely to recur, and have a trigger/check.
3. Merge duplicates with existing `Orders` entries.
4. Add or update a `Refinements` entry in `.agents/Learning/Learning.md`.
5. Retire stale lessons instead of silently deleting them.
