# Imported GacUI Agent Context

Use it to study the GacUI context-system pattern:

- top-level agent instruction entrypoint;
- guidelines for build/test/debug workflows;
- knowledge base organization;
- learning and task-log structure;
- prompt and agent-tool organization.

Symphony's canonical agent guidance remains in `.agents/`.
